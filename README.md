# 🎬 Movie Information Extractor

This is a Streamlit web application that extracts structured movie information from a paragraph using **LangChain** and **Mistral AI**.

## Features

- Extracts movie title
- Release year
- Genre
- Director
- Cast
- Rating
- Summary

## Installation

```bash
pip install -r requirements.txt
```
##url
https://movie-info-extractorr-3pbcut3pfnoxnxrzejfocr.streamlit.app/

## Run Locally

```bash
streamlit run app.py
```

## Environment Variable

Create a `.env` file and add:

```text
MISTRAL_API_KEY=your_api_key_here
```
